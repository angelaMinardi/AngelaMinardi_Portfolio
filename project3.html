AI-Powered Chatbot

I developed a chatbot that uses machine learning techniques to understand and respond to user inputs in a conversational manner. The chatbot serves as a personal assistant, capable of answering questions and performing simple tasks.

Key Features:

  • Natural Language Processing using NLTK and spaCy
  • Machine learning models developed with TensorFlow
  • Integrated with a web-based user interface
  • Supports multiple languages and intents

Technologies Used:

  • Python programming language
  • TensorFlow and Keras for machine learning
  • Flask for web framework
  • JavaScript for frontend interactivity
