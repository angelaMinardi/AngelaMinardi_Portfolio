Smart Insole Technical Design

In this project, I developed an intelligent smart insole using ESP32 microcontrollers, force-sensing resistors, and inertial measurement units (IMUs) to monitor gait patterns and foot pressure distribution in real-time.

I produced detailed technical documentation, including design specifications, circuit diagrams, firmware documentation, and user manuals. The documentation aimed to make the complex technical aspects accessible to both technical and non-technical stakeholders.

Key Features:

  • Real-time gait analysis and fall prevention mechanisms
  • Data transmission using MQTT protocol
  • Integration with AWS for data processing and storage
  • Comprehensive technical documentation and user guides

Technologies Used:

  • ESP32 Microcontroller
  • C++ and Arduino IDE for firmware development
  • MQTT Protocol for data transmission
  • AWS IoT Core and AWS Lambda
  • Markdown and LaTeX for documentation
