5G Wireless Network Simulation

In this project, I developed a simulation tool to model and analyze the performance of 5G wireless networks using NS-3. The simulator allows users to configure network parameters and visualize how changes affect network performance, focusing on key metrics such as throughput, latency, and packet loss.

Key Features:

  • Simulated gNB (5G base stations) and UEs (User Equipment)
  • Implemented mobility models for realistic movement of UEs
  • Analyzed performance metrics under varying network conditions
  • Customizable network topologies and traffic patterns

Technologies Used:

  • C++ programming language
  • NS-3 network simulator
  • Gnuplot for data visualization
  • Git for version control
