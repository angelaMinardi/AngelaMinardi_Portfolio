Analyzing Reading Proficiency Trends

This project involves analyzing reading proficiency trends among students using datasets from the National Assessment of Educational Progress (NAEP). The analysis focuses on understanding how reading scores have evolved over time and across different demographic groups. The project demonstrates skills in data cleaning, statistical analysis, data visualization, and deriving actionable insights to support educational interventions.

Key Features:

  • Data exploration and cleaning using Python's Pandas library
  • Descriptive and inferential statistical analysis
  • Interactive data visualizations with Chart.js
  • Comprehensive reporting and insights generation

Technologies Used:

  • Python (Pandas, NumPy, SciPy, Matplotlib)
  • JavaScript (Chart.js)
  • HTML5 & CSS3
  • Prism.js for code syntax highlighting

// Example: Data Cleaning and Preparation
import pandas as pd

# Load the data
data_2023 = pd.read_excel("2023_appendix_ltt_13.xlsx", sheet_name="RP-Performance-RED-a13")
data_2022 = pd.read_excel("2022_appendix_ltt_9.xlsx", sheet_name="RP-Performance-RED-a09")

# Clean the data
data_2023_cleaned = data_2023.iloc[3:].dropna(thresh=5).reset_index(drop=True)
data_2022_cleaned = data_2022.iloc[3:].dropna(thresh=5).reset_index(drop=True)

# Rename columns
columns_2023 = ["Characteristics"] + [str(year) for year in [1971, 1975, 1980, 1984, 1988, 1990, 1992, 1994, 1996, 1999, "2004_1", "2004_2", 2008, 2012, 2020, 2023]]
data_2023_cleaned.columns = columns_2023

columns_2022 = ["Characteristics"] + [str(year) for year in [1971, 1975, 1980, 1984, 1988, 1990, 1992, 1994, 1996, 1999, "2004_1", "2004_2", 2008, 2012, 2020, 2022]]
data_2022_cleaned.columns = columns_2022

# Convert data types
numeric_columns = data_2023_cleaned.columns[1:]
data_2023_cleaned[numeric_columns] = data_2023_cleaned[numeric_columns].apply(pd.to_numeric, errors='coerce')
data_2022_cleaned[numeric_columns] = data_2022_cleaned[numeric_columns].apply(pd.to_numeric, errors='coerce')

Insights and Recommendations

  • Trend Analysis: Average reading scores have shown a consistent improvement over the years, indicating positive impacts of educational interventions.
  • Demographic Disparities: Significant differences exist among demographic groups, with certain groups (e.g., Black, Hispanic) trailing in reading proficiency, highlighting the need for targeted support.
  • Data Imputation Impact: Handling missing values through imputation maintained the integrity of statistical analyses, ensuring reliable insights.
  • Recommendations for Ignite Reading:
    • Implement targeted interventions for demographic groups with lower reading scores.
    • Continuous monitoring of student progress through real-time data tracking.
    • Allocate resources strategically to bridge the achievement gap among underrepresented groups.
    • Conduct further qualitative research to understand underlying factors affecting reading proficiency.
