Analyzing Reading Proficiency Trends Using NAEP Data

This project involves analyzing reading proficiency trends among students using datasets from the National Assessment of Educational Progress (NAEP). The analysis focuses on understanding how reading scores have evolved over time and across different demographic groups. The insights derived aim to support educational interventions and policies to improve reading outcomes.

Key Features:

  • Data Cleaning and Preparation using Python and Pandas
  • Descriptive and Inferential Statistical Analysis
  • Interactive Data Visualizations with Chart.js
  • Insightful Reporting and Recommendations

Technologies Used:

# Data Cleaning and Preparation
import pandas as pd

# Load datasets
data_2023 = pd.read_excel('2023_appendix_ltt_13.xlsx', sheet_name='RP-Performance-RED-a13', skiprows=3)
data_2022 = pd.read_excel('2022_appendix_ltt_9.xlsx', sheet_name='RP-Performance-RED-a09', skiprows=3)

# Rename columns
data_2023.columns = ["Characteristics", "1971", "1975", "1980", "1984", "1988", "1990", "1992",
                     "1994", "1996", "1999", "2004_1", "2004_2", "2008", "2012", "2020", "2023"]

data_2022.columns = ["Characteristics", "1971", "1975", "1980", "1984", "1988", "1990", "1992",
                     "1994", "1996", "1999", "2004_1", "2004_2", "2008", "2012", "2020", "2022"]

# Drop rows with insufficient data
data_2023.dropna(thresh=5, inplace=True)
data_2022.dropna(thresh=5, inplace=True)

# Reset index
data_2023.reset_index(drop=True, inplace=True)
data_2022.reset_index(drop=True, inplace=True)

# Convert columns to numeric
numeric_cols = data_2023.columns[1:]
data_2023[numeric_cols] = data_2023[numeric_cols].apply(pd.to_numeric, errors='coerce')

numeric_cols_2022 = data_2022.columns[1:]
data_2022[numeric_cols_2022] = data_2022[numeric_cols_2022].apply(pd.to_numeric, errors='coerce')
            

# Descriptive Statistics
avg_scores_2023 = data_2023.mean(numeric_only=True)
avg_scores_2022 = data_2022.mean(numeric_only=True)

# Plotting Average Scores Over Time
import matplotlib.pyplot as plt

plt.figure(figsize=(14, 7))
plt.plot(avg_scores_2023.index, avg_scores_2023.values, marker='o', linestyle='-', label='2023 Data (Age 13)')
plt.plot(avg_scores_2022.index, avg_scores_2022.values, marker='o', linestyle='--', label='2022 Data (Age 9)')

plt.xlabel('Year')
plt.ylabel('Average Reading Score')
plt.title('Trends in Average Reading Scores Over Time')
plt.xticks(rotation=45)
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

Insights and Recommendations

  • Trend Analysis: Average reading scores have shown a steady increase over the years, indicating improvements in reading proficiency.
  • Demographic Disparities: Significant differences exist among demographic groups, with certain groups (e.g., Black, Hispanic) showing slower progress.
  • Targeted Interventions: Focused support for underperforming groups can help bridge the achievement gap and promote equity in education.
  • Continuous Monitoring: Implement real-time data tracking to monitor student progress and adjust strategies promptly.
