5G Wireless Network Simulation

This project involves the development of a simulation tool using NS-3 to model and analyze 5G wireless networks. The focus was on key performance indicators such as throughput, latency, and packet loss.

I authored comprehensive technical documentation, including user manuals and technical reports, to facilitate understanding and usage of the simulation tool by other researchers and professionals.

Key Features:

  • Simulated 5G network scenarios with varying parameters
  • Analyzed network performance metrics
  • Developed user guides and technical documentation
  • Collaborated with a team to validate simulation results

Technologies Used:

  • NS-3 Network Simulator
  • C++ Programming Language
  • Gnuplot for data visualization
  • Markdown and LaTeX for documentation
