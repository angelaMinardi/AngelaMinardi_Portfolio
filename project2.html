Blockchain Implementation

In this project, I built a simplified blockchain from scratch to understand the fundamentals of distributed ledger technology. The implementation includes core features like proof of work, hashing, and transaction validation.

I documented the entire development process, explaining the blockchain architecture, algorithms, and implementation details to serve as an educational resource for others interested in blockchain technology.

# Example: Proof-of-Work Algorithm
import hashlib
import json

def proof_of_work(last_proof):
    proof = 0
    while valid_proof(last_proof, proof) is False:
        proof += 1
    return proof

def valid_proof(last_proof, proof):
    guess = f'{last_proof}{proof}'.encode()
    guess_hash = hashlib.sha256(guess).hexdigest()
    return guess_hash[:4] == "0000"
