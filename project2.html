Blockchain Implementation

This project involved building a simplified blockchain from scratch to understand the underlying mechanisms of distributed ledger technology. The blockchain supports basic functionalities such as adding transactions, mining blocks, and validating the chain.

Key Features:

  • Implemented Proof of Work consensus algorithm
  • Used cryptographic hashing for block security
  • Simulated a network of nodes for decentralization

Technologies Used:

  • Python programming language
  • Flask for web interface (optional)
  • Hashlib library for hashing functions
