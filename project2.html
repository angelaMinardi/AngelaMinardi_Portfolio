Blockchain Implementation

This project involved building a simplified blockchain from scratch to understand the underlying mechanisms of distributed ledger technology. The blockchain supports core functionalities such as adding transactions, mining blocks, and validating the chain.

Key Features:

  • Implemented Proof of Work consensus algorithm
  • Used cryptographic hashing for block security
  • Simulated a network of nodes for decentralization
  • Developed transaction validation and chain integrity checks

Technologies Used:

  • Python programming language
  • Flask for web interface
  • Hashlib library for hashing functions
  • Git for version control

// Example: Proof of Work Implementation
import hashlib
import time

class Block:
    def __init__(self, index, transactions, timestamp, previous_hash, nonce=0):
        self.index = index
        self.transactions = transactions
        self.timestamp = timestamp
        self.previous_hash = previous_hash
        self.nonce = nonce
        self.hash = self.compute_hash()
    
    def compute_hash(self):
        block_string = str(self.__dict__)
        return hashlib.sha256(block_string.encode()).hexdigest()
    
    def proof_of_work(self, difficulty):
        self.nonce = 0
        computed_hash = self.compute_hash()
        while not computed_hash.startswith('0' * difficulty):
            self.nonce += 1
            computed_hash = self.compute_hash()
        self.hash = computed_hash
        return computed_hash
