Smart-Insole Technical Design

Falls among geriatric populations represent a significant public health challenge, leading to serious injuries, increased mortality rates, and substantial economic burdens on healthcare systems and families. To address this critical issue, I have developed an innovative smart insole to monitor and analyze gait patterns and foot pressure distribution in real-time, thereby predicting and preventing potential falls.

Key Features:

  • Real-time gait monitoring using advanced sensors
  • Predictive analytics to assess fall risk
  • Secure data transmission to cloud services
  • Comprehensive user interface with actionable insights
  • Optimized power management for extended battery life

Technologies Used:

  • Hardware: ESP32, Force Sensitive Resistors (FSRs), MPU9250 IMU
  • Firmware: C++ (Arduino framework)
  • Cloud Services: AWS IoT Core, AWS SageMaker
  • Machine Learning: XGBoost
  • Frontend: React Native (Smartphone Application)
  • Protocols: MQTT over TLS

Technical Design

System Overview

The Smart-Insole integrates advanced sensors, including a Force Sensitive Resistor (FSR) matrix and a 9-axis Inertial Measurement Unit (IMU), which work in tandem with a powerful ESP32 microcontroller to collect and process comprehensive biomechanical data.

Hardware Design

FSR Matrix Selection and Configuration

Selected thin-film FSRs (Interlink FSR 402 series) for accurate pressure mapping, arranged in a 4x8 grid to cover critical regions of the foot: Heel, Midfoot, Forefoot, and Toes.

Electrical Circuit Design and Multiplexing

Each FSR is incorporated into a voltage divider circuit to convert resistance changes into voltage signals suitable for analog-to-digital conversion. Utilized analog multiplexers to efficiently manage multiple sensor inputs.

Integration of the Inertial Measurement Unit (IMU)

Integrated the MPU9250 IMU for comprehensive 9-axis motion tracking, centrally placed within the insole to minimize rotational artifacts and ensure accurate measurement of gait dynamics.

ESP32 Microcontroller and Connectivity

The ESP32 microcontroller facilitates data collection, processing, and secure transmission to AWS IoT Core via MQTT over TLS, ensuring real-time analytics and fall risk prediction.

Power Supply and Battery Life Estimation

Implemented power management strategies, including deep sleep modes and duty cycling, to extend battery life to approximately three days under typical usage patterns.

Data Acquisition and Processing

Sampling Rates and Synchronization

Configured uniform sampling rates for FSR and IMU sensors at 100 Hz to facilitate synchronized data fusion and accurate gait analysis.

Feature Extraction

Extracted critical gait parameters such as Center of Pressure (CoP), stride length, and postural stability measures. Employed XGBoost for predictive analytics to assess fall risk based on these features.

Cloud Integration and Security

Utilized AWS IoT Core for secure device connectivity and data ingestion. Deployed the XGBoost model on AWS SageMaker for scalable and efficient fall risk prediction. Ensured data security and compliance with HIPAA and GDPR through encryption and robust access controls.

User Interface and Feedback

Developed a responsive smartphone application using React Native, providing users with real-time fall risk scores, historical data visualizations, and personalized recommendations. Incorporated accessibility features to ensure usability across diverse user populations.

Prototype Development and Testing

Physical Integration and Insole Design

Constructed the insole using flexible, breathable materials such as EVA foam and TPU for durability. Employed finite element analysis (FEA) to simulate mechanical stresses and optimize component placement for longevity and user comfort.

Model Deployment on AWS

Deployed the XGBoost model on AWS SageMaker endpoints, enabling real-time inference based on incoming sensor data. Implemented secure MQTT communication to ensure data integrity and privacy during transmission.

User Testing and Feedback

Conducted pilot studies with elderly participants to evaluate system performance, user acceptance, and adherence rates. Gathered feedback to refine the user interface and enhance overall system usability.

Conclusion

The Smart-Insole project represents a significant advancement in fall risk assessment technology for older adults. By integrating high-resolution pressure sensing with comprehensive motion analysis and sophisticated machine learning algorithms, the system provides accurate, real-time predictions of fall risk. The design choices are meticulously justified based on technical merits, user needs, and practical considerations, ensuring the device is effective and user-friendly.
